# SkillBridge-AI-AWS

An AI-powered internship recommendation and career guidance platform built using AWS serverless architecture.

## AWS Services Used

- Amazon Bedrock (Claude 3 Haiku)
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- Amazon S3
- Amazon Cognito (planned)

## Features

- Resume skill extraction
- Internship compatibility scoring
- AI career chatbot
- Success probability explanation
